Project 1: : Garbage Monitoring System

Description: The Garbage Monitoring System combines an HC-05 Bluetooth module and an ultrasonic sensor to measure garbage levels. As the garbage fills up, the sensor detects the change in distance and sends this data wirelessly to the Blynk cloud. Users can then easily track and manage garbage levels in real-time through the Blynk app or web interface, streamlining waste management. .
